The Pinterest pin description field accepts **800 characters**, but only the **first ~50 characters render in the mobile feed** before a "…more" cutoff. On desktop the visible window is wider — roughly 200 characters — but mobile is where the majority of Pinterest impressions happen, so plan to the ~50-character feed window. This applies to organic Pins and Promoted (ad) Pins identically; the description field and its truncation point do not change when a Pin is promoted.

Pinterest treats the description differently from every other social platform. It is not in-feed persuasion copy — feed users barely see it. It is a **search-and-match field**: Pinterest's visual search engine and Google image search both read the full 800 characters to decide what queries the Pin should rank for. So the first ~50 characters must read as a complete, compelling thought for the minority who read in-feed, while the remaining ~750 should be dense, natural-language long-tail keyword coverage for the majority who arrive through search.

## How to use the 800 characters

- **First ~50 characters: a complete value statement.** "Cream merino crew sweater, machine washable, under $80" reads fully in-feed and as the search snippet.
- **Next ~750 characters: long-tail keyword body.** Cover the synonyms, use cases, and qualifiers a searcher would type — "fall capsule wardrobe", "non-itchy wool", "gift for her". This is what makes the Pin match queries; it is not wasted space.
- **Do not keyword-stuff.** Pinterest's NLP rewards natural phrasing. Write sentences a human would write; the keyword coverage should fall out of describing the product thoroughly.
