Pinterest Carousel Ads stack 2 to 5 cards in a single Pin slot. Each card has its own title (100 chars / ~40 visible) and description (800 chars / ~50 visible mobile, ~200 visible desktop). Carousel cards swipe horizontally inside the same pin tile, with a small dot indicator at the bottom showing position.

Unlike Meta carousel ads, Pinterest carousel cards do not change the destination URL per card by default — all 2-5 cards point to the same destination unless explicitly overridden. The format is best suited for "shown 5 ways" or "step 1 through 5" narratives where the cards serve a single underlying concept.

## Per-card destination

The default is single-destination across all cards. To run per-card destination URLs, configure each card individually at upload time — useful for category-level carousels where each card pushes to a different product page.
