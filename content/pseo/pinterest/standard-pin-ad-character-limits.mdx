Pinterest Standard Pin Ads are the platform's default format: a single 2:3 vertical image with a title and description rendered below. The title field allows up to 100 characters but only the first ~40 render on mobile feed before truncation by the bottom CTA strip. The description allows up to 800 characters but the visible window on mobile is roughly 50 characters before "...more" — desktop is more generous at ~200 visible characters.

Pinterest's strategic constraint is the inverse of every other social platform: the image does the entire selling job. Pinterest users are in active "save for later" mode, not feed-scroll-and-react mode. The text fields exist for SEO and rich-results matching, not for in-feed copy persuasion. Plan the title as a search-optimized keyword phrase and the description as a paragraph of long-tail keywords — visitors who tap into the pin detail view will read more than visitors anywhere else.

## SEO-driven copy

Pinterest is closer to a search engine than a social feed in user intent. Title text functions as the keyword phrase; description functions as the long-form keyword body. Generic copy like "Beautiful sweater for fall" is wasted; specific copy like "Cream merino crew neck sweater for fall under $80" hits multiple search clusters.
