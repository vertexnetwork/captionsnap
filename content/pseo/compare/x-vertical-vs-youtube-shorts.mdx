X Vertical Video Ads and YouTube Shorts Ads use nearly identical UI vocabulary — full-bleed 9:16 video, right-rail engagement column, bottom-left username block, sponsored CTA strip — but with different visible caption windows.

X Vertical Video clips post text at roughly 100 visible characters. YouTube Shorts clips description at roughly 60. The difference traces to the right-rail icon count: X has four icons (reply, repost, like, bookmark), YouTube has six (like, dislike, comment, share, remix, subscribe), so YouTube's UI claims more pixels for itself and leaves less for the caption.

Plan to YouTube's tighter clamp when running cross-platform vertical video creative. A 60-character hook works on both; a 100-character X-optimized hook will get clipped on Shorts.
